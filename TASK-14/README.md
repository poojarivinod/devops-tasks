#  Event-Driven Automation using S3, Lambda & CloudWatch 
## Concept Overview 
This setup demonstrates a serverless automation pipeline where an AWS Lambda function is 
triggered automatically whenever a file is uploaded to an Amazon S3 bucket. 
You don’t have to: 
 Run any code manually 
 Manage or maintain servers 
Everything works automatically in response to events — that’s called Event-Driven Architecture.
